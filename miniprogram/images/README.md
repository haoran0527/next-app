# 微信小程序图标说明

小程序需要以下图标文件，请将图标放置在 `miniprogram/images/` 目录下：

## 必需的图标文件

### TabBar 图标 (81x81 px)
- `home.png` - 首页图标（未选中）
- `home-active.png` - 首页图标（选中）
- `add.png` - 记账图标（未选中）
- `add-active.png` - 记账图标（选中）
- `list.png` - 明细图标（未选中）
- `list-active.png` - 明细图标（选中）
- `stats.png` - 统计图标（未选中）
- `stats-active.png` - 统计图标（选中）
- `profile.png` - 我的图标（未选中）
- `profile-active.png` - 我的图标（选中）

## 图标设计建议

1. **风格**：使用简洁的线性图标或填充图标
2. **颜色**：
   - 未选中：灰色 (#999999)
   - 选中：蓝色 (#1890ff)
3. **尺寸**：81x81 像素
4. **格式**：PNG 格式，支持透明背景

## 临时解决方案

在准备图标之前，可以使用以下方式：

1. 使用微信开发者工具自带的图标
2. 使用在线图标库（如 iconfont）下载图标
3. 使用设计工具（如 Figma、Sketch）创建图标

## 快速获取图标

可以从以下网站免费获取图标：
- Iconfont (https://www.iconfont.cn/)
- Flaticon (https://www.flaticon.com/)
- IconPark (https://iconpark.oceanengine.com/)
